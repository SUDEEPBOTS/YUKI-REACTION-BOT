## 📌 𝗔𝗯𝗼𝘂𝘁 𝗧𝗵𝗲 𝗕𝗼𝘁

**YUKI REACTION BOT** is a custom-built Telegram bot designed to instantly react to messages in groups and channels. By utilizing user accounts (Multi-Client support), it ensures that every post gets engagement automatically without any manual effort.

---

## 🔥 𝗙𝗲𝗮𝘁𝘂𝗿𝗲𝘀

- ✅ **Instant Reactions**: Drops emojis the second a message is sent.
- ✅ **Multi-Session Support**: Run 5, 10, or even 20 accounts simultaneously.
- ✅ **Media Specific**: Reacts differently to text, photos, and videos.
- ✅ **Anti-Ban Mechanics**: Smart delays to prevent Telegram spam limits.
- ✅ **Database Backed**: Saves settings in MongoDB so you never lose them.

### 💻 𝗩𝗣𝗦 / 𝗟𝗼𝗰𝗮𝗹 𝗗𝗲𝗽𝗹𝗼𝘆

**1. Clone the Source Code**
```bash
git clone https://github.com/SUDEEPBOTS/YUKI-REACTION-BOT.git
cd YUKI-REACTION-BOT
```

**2. Install Dependencies**
```bash
pip3 install -U -r requirements.txt
```

**3. Set up Environment Variables**
Create a `.env` file and add the following details:
```env
API_ID=your_api_id
API_HASH=your_api_hash
BOT_TOKEN=your_bot_token
MONGO_DB_URI=mongodb+srv://...
OWNER_ID=your_telegram_id
```

**4. Start the Bot**
```bash
python3 -m YUKIREACTION
```
